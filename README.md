# 🔑 Key Press Viewer

A simple and stylish web app that shows the key you press, its code, and the key code.

---

## 🚀 How to Run

Open `index.html` in your browser.

---

## 🧠 Technologies Used

- HTML5  
- CSS3 (with Glassmorphism design)  
- Vanilla JavaScript
